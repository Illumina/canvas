RNAExpress Summary Report Description

Alignment Summary

This table contains statistics describing the alignment of reads to the genome.
  • Sample: The name of the sample (all replicates of one condition have the same sample name).
  • Replicate: The name of the individual replicate.
  • Read Count: The number of reads (or pairs of reads for paired-end sequencing runs).
  • Uniquely mapped: The percentage of reads that have a unique best alignment position in the genome
  • Multi-mapped: The percentage of reads that have more than one equally good alignment positions in the genome
  • Average mapped length: The average number of bases that are aligned to the genome for each read (pair of reads for PE runs). The difference between this number and the read-length is due to trimming of unaligned sequences (and adapters) at the ends of reads.
  • Mismatch rate: The percent of positions within the aligned portion of a read that do not match the reference genome sequence
  • Number of splices: The total number of splicing-events. Each case where a read-alignment jumps over a known or discovered intron is counted.
  • Alignments spliced: The percentage of aligned reads that contain at least one splice-site (i.e. span at least one intron)

Abundant Sequences

This table lists the percentage of all reads that align to various abundant non-mRNA annotation types in the target genome
  • Adapter refers to the percentage of reads that have a match to a sequencing adapter. These adapter matches are trimmed off and the remaining sequence of the read is used for further analysis.

Read Counts

This table contains statistics describing the expression quantification, i.e. counting reads per gene.
  • Assigned reads: The percentage of totals reads (or read pairs for PE runs) that were aligned to a gene. These reads were uniquely mapped and overlapped exons of one (only one) gene.
  • Unassigned reads: Aligned reads that could not be assigned to any gene, i.e. mapping outside of the annotated exons
  • Ambiguous: Reads that overlap exons from more than one gene
  • Number of genes: The number of genes in the reference annotation file used

Differential Expression

This table summarizes the results of any requested differential expression analyses.
  • Control: The control sample
  • Comparison: The comparison sample
  • Total Genes: The total number of genes evaluated in the comparison
  • Tested Genes: The number of genes for which a complete differential expression test was performed (not filtered for e.g. low expression).
  • Differential Genes: The number of significantly differentially expressed genes

Differential Gene Browser

An interactive browser providing an overview of the differential expression analysis results. The scatter-plot shows the mean expression level on the x-axis (in read counts) and the estimated log fold-change between the two conditions for each gene. The controls on the left allow to select subsets of all genes.
  • Status: Genes with status LOW were not tested, due to low average expression across samples. OUTLIER marks genes for which the result is strongly affected by a single (outlier) replicate in either sample.
  • Significant: Genes with an multiple-testing adjusted p-value for differential expression of less than 0.05
The table shows differential expression results for each gene currently selected.
  • Mean Count: The mean read count for this gene across all samples
  • log(Fold Change): Estimated log fold change between the two samples
  • Std. err. log(Fold Change): Standard error of the Fold Change estimate
  • q value: Multiple-testing adjusted p-value for differential expression
  • M/A Plot: Static Mean vs. fold-change plot for all genes. Red points are significant, outliers are drawn as triangles pointing out of the plot-area.
  • Dispersion estimates: Plot showing the relationship between mean expression level of all genes and their variability (dispersion) as estimated by DESeq2. The red line marks the average relationship, the blue dots the final values used for each gene.
  • Gene heatmap: Log(expression) of differentially expressed genes across all replicates ordered by similarity.

Sample Correlation

A heatmap showing the relative similarity between all samples and replicates in this analysis-run. Each row and column represents one replicate, ordered by similarity (hierarchical clustering). The color of each field indicates the correlation between these replicates (Spearman Rho).

PCA Plot

A scatter plot showing the relative distance between all samples and replicates in this analysis-run. The regularized log-transformed counts are used for principal component analysis.
